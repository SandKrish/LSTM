# Deutsche Bank Stock Prediction using LSTM

This project uses **Long Short-Term Memory (LSTM)** neural networks to analyze and predict the stock price of **Deutsche Bank**.

The notebook analyzes stock data from:

* EQUITY – New York
* Germany
* France

The model is trained to **predict the closing stock price of Deutsche Bank (EQUITY – New York)** using historical data.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* LSTM (Deep Learning)
